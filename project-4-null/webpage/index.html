Overall story

Data analysis of possible factors affecting people's willingness of COVID-19 vaccination.

The COVID-19 has affected the economy and people’s lives across the globe.Vaccines can effectively reduce the spread of COVID-19. Therefore, we want to find out what factors around our lives can affect people’s willingness to get vaccinated, which might provide more ideas to help the government to find better ways to reduce the spread of the epidemic.

We collected data from four datasets, which contain information of vaccination, the number of new cases by time, the usage of public resources to reduce the spread, and people's sentiment on tweets about COVID-19. We did some preprocess on these data, like data cleaning and bin. Then we used some statistical analysis method and machine learning method to analyze this data. We made several hypotheses, used some of this data to analyze what features would influence people' willingness to get vaccinated. At last, we found there is a positive correlation between epidemic severity, the usage of public resources and people’s willingness to get vaccinated.Hope our analysis can make some little contributions to promoting people's willingness to vaccinate
