Network Graph

Network Details

Each node represents one state. If the difference between the hospitial capacity of the state A and state B is within a specific value, then regard them as similiar states and draw a edge from node A to node B. The weight of each edge is determined by the differnece of two nodes' williness.This network is bulit on one specific date.

Date: 2021-10-14

Number of nodes: 49

Number of edges: 269

Network density: 0.229

Cluster coefficient: 0.761

Analysis

Based on this network, we can split different states into several groups by their willingness. Importantly, we find the hosipital capacity may have effect on people's effect, thus, when we are building these network, we only connect two states which have similiar hosipital capacity. Some basic information like network density, each node's betweeness and some other information is useless for this topic. Those information focus on each state's hospitial situation instead of vaccination willingness. The only useful infromation is the network's clusters. After clustering, the cluster coefficient is 0.76 which means the cluster performance is relatively good. Based on the result, we can find simliar states which have simliar willingness regardless of the effect of hospitial capacity on williness. If there are some common factors among those states, that factor might have effect on willingness. For example, maybe some of those states release a same policy one day, after a few days, these states are splited into the same cluster, then we can regard that the policy has some effect. However, we did't collect any data about the policy or some relative factors. In the future, we will focus on it.
